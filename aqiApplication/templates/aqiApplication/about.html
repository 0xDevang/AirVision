About AQI Monitor

Your Simple Air Quality Tracking System

AQI Monitor is a lightweight and fast web application designed to provide real-time Air Quality Index (AQI) data for major Indian cities. The goal of this platform is to make essential environmental information easily accessible to everyone without any complexity.

What This Website Does:

  • Displays real-time AQI values using the WAQI (World Air Quality Index) API.
  • Shows the dominant pollutant affecting the air quality in each city.
  • Presents the latest update timestamp for transparency.
  • Allows users to search any city directly from the navigation bar.
  • Provides an overview of AQI for several major Indian cities in one place.

Why Air Quality Matters:

Air pollution has become one of the most critical environmental and public health challenges in modern cities. Monitoring AQI helps individuals make informed decisions about outdoor activities, health precautions, and general well-being. Our purpose is to give you this information instantly and clearly.

Technology Used:

  • Django for backend logic and routing
  • HTML + CSS for clean front-end UI
  • WAQI API for fetching real-time air data
  • Python Requests for API communication

Our Vision:

The vision behind AQI Monitor is to create a simple, fast, and reliable interface to help users stay aware of the air they breathe. In future versions, more features such as live AQI maps, historical trends, graphs, and notifications may be added.
